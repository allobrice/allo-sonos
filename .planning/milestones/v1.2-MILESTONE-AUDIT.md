## E2E Flow (1/1 complete)

**Flow: Open favorites → Select → Playback starts → Sheet closes**

16-step trace from user tap to WebSocket state update — all steps wired:

1. Heart button visible on online zones (v-if="!isOffline")
2. Tap → toggleFavorites() → showFavorites = true
3. Sheet opens with slide-up Transition
4. Watch(visible) → loadFavorites() → GET /api/favorites
5. Backend SOAP Browse FV:2 → parse DIDL-Lite → return array
6. Favorites rendered with type icons
7. Tap favorite → handleSelect()
8. playFavorite() → POST /api/speakers/:id/play-favorite
9. Backend: speaker lookup → coordinator routing → SetAVTransportURI + Play
10. emit('close') → sheet closes immediately
11. WebSocket state_changed → ZoneCard updates asynchronously

**Verdict: COMPLETE — no breaks in the E2E flow.**
