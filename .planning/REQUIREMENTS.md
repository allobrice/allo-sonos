# Requirements: Sonos Pilot

**Defined:** 2026-03-02
**Core Value:** Contrôler la musique de n'importe quelle zone en moins de 3 secondes, sans friction ni surcharge visuelle.

## v1.2 Requirements

Requirements for milestone v1.2 Sonos Favorites. Each maps to roadmap phases.

### Favoris

- [ ] **FAV-01**: User peut voir la liste de tous les favoris Sonos depuis une zone card
- [ ] **FAV-02**: User peut lancer un favori (station, playlist, album) sur la zone sélectionnée
- [ ] **FAV-03**: User voit le type de favori (station radio, playlist, album) dans la liste
- [ ] **FAV-04**: Les favoris se chargent depuis le système Sonos (ContentDirectory SOAP)

### Navigation

- [ ] **NAV-01**: User peut ouvrir/fermer le panneau favoris via un bouton sur la ZoneCard
- [ ] **NAV-02**: La liste des favoris se ferme automatiquement après sélection

## Future Requirements

### Spotify Integration (v1.3+)

- **SPOT-01**: Intégration Spotify API directe (OAuth)
- **SPOT-02**: Recherche musicale Spotify (artiste, album, morceau)
- **SPOT-03**: Parcourir et lancer des playlists Spotify

### File d'attente (v1.3+)

- **QUEUE-01**: Voir la file d'attente de la zone
- **QUEUE-02**: Réordonner les morceaux dans la file
- **QUEUE-03**: Supprimer des morceaux de la file

### Groupes (v1.4+)

- **GRP-01**: Grouper/dégrouper des zones depuis l'UI

## Out of Scope

| Feature | Reason |
|---------|--------|
| Gestion avancée des groupes | Reportée à v1.4+ |
| API Spotify directe | Reportée à v1.3+ — commencer par favoris Sonos natifs |
| Recherche musicale cross-service | Trop ambitieux pour ce milestone |
| Comptes utilisateurs individuels | Un PIN partagé suffit |
| Accès externe (hors LAN) | Surface d'attaque, l'équipe est sur site |
